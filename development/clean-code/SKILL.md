# Clean Code Skill

This skill provides access to the official Clean Code guidelines. Use the `consult_clean_code` tool to retrieve the full set of principles when in doubt.

## Core Principles

- **SRP**: Single Responsibility
- **DRY**: Don't Repeat Yourself
- **KISS**: Keep It Simple
- **YAGNI**: You Aren't Gonna Need It
- **Boy Scout**: Leave code cleaner than you found it

## Verification

Always verify your changes against these principles.
